About the Project

Hand & Gesture Tracker

This application uses Google MediaPipe's cutting-edge vision tasks resolver engine to carry out real-time multi-hand landmark tracking and hand gesture classification directly in your browser.

Sub-5ms Inference

Throttled frame processing running outside Angular's zone to prevent CPU/GPU thermal limits while preserving 60 FPS drawing overlays.

Local Sandbox Privacy

All camera streams, files, and static uploads are processed entirely locally. No video or image data is ever uploaded to external servers.

Glassmorphism UI

Crafted using HSL design palettes, translucent frosted overlays, smooth CSS transitions, and distinct custom neon landmarks.

Technology Stack

MediaPipe Tasks-Vision Ionic Standalone v8 Angular v20 WebAssembly SIMD TypeScript SCSS
